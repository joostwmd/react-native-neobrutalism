# Collapsible

An interactive component that expands/collapses a panel with smooth animation.

## Import

```tsx
import {
  Collapsible,
  CollapsibleTrigger,
  CollapsibleContent,
} from 'react-native-neobrutalism';
```

## Basic Usage

```tsx
<Collapsible>
  <CollapsibleTrigger>Click to expand</CollapsibleTrigger>
  <CollapsibleContent>
    <Text>This is the hidden content.</Text>
  </CollapsibleContent>
</Collapsible>
```

## Controlled State

Control the open state externally.

```tsx

  <CollapsibleTrigger>Controlled</CollapsibleTrigger>
  <CollapsibleContent>
    <Text>Controlled content.</Text>
```

## Props

### Collapsible

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Default open state (uncontrolled) |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when state changes |
| `disabled` | `boolean` | `false` | Disable interaction |
| `themeOverride` | `NeobrutalismThemeOverride` | - | Theme overrides |
| `style` | `ViewStyle` | - | Container styles |

### CollapsibleTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Trigger content |
| `asChild` | `boolean` | `false` | Clone child instead of wrapping |
| `showChevron` | `boolean` | `true` | Show chevron indicator |
| `chevronIcon` | `ReactNode` | - | Custom chevron icon |
| `style` | `ViewStyle` | - | Container styles |
| `textStyle` | `TextStyle` | - | Text styles |
| `accessibilityLabel` | `string` | - | Accessibility label |

### CollapsibleContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Content to show/hide |
| `forceMount` | `boolean` | `false` | Keep mounted when closed |
| `style` | `ViewStyle` | - | Container styles |

## Animation

The collapsible uses smooth height animation via React Native Reanimated:
- Content height is measured automatically
- Chevron rotates 180 degrees when expanded
- Animation duration follows theme settings

## Accessibility

- Trigger uses `accessibilityRole="button"`
- `accessibilityState.expanded` reflects open state
- `accessibilityState.disabled` when disabled
- Content is hidden from screen readers when collapsed
